# Analytics-practice

This repository contains practice projects and exercises related to data analytics.  
The goal is to practice different tools, data cleaning, and basic data visualization.

## Contents

### Power BI
Practice dashboards focused on data visualization and dashboard layout.

### Excel / PowerPivot
Practice exercises for preparing and visualizing data in Excel and PowerPivot.

### Google Sheets
Practice dashboards built using Google Sheets for basic data analysis and visualization.

### SQL
SQL exercises for querying, filtering, and aggregating data.

## Tools
- Power BI
- Excel / PowerPivot
- Google Sheets
- SQL
